# Linux Zero to Hero

Master Linux system administration - the foundation of all DevOps work.

## What You'll Learn

- Linux file system and directory structure
- Command line essentials (bash, zsh)
- File permissions and ownership
- Process management
- System services and systemd
- Package management (apt, yum, dnf)
- Networking and firewall configuration
- Log management and monitoring
- Shell scripting basics
- Performance tuning and troubleshooting

## Prerequisites

- Basic computer skills
- Willingness to learn command line

## Why Linux for DevOps?

Linux powers most servers, containers, and cloud infrastructure. Understanding Linux is essential for:
- Managing servers and infrastructure
- Working with containers (Docker, Kubernetes)
- Cloud platforms (AWS, Azure, GCP)
- CI/CD pipelines
- Automation and scripting
